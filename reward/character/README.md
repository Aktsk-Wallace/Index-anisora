## Getting Started

### Download Checkpoints

Please download the checkpoints from [there]() and put them in `./IP_checkpoints/`. It contains model that we pretrained and some open-source models such as blip, sam2.1 and grounding-dino checkpoints.

### Data Preparation

Please download the gallery data from [there]() and put them in `./videos/`:
```
/reward/character/
├── BLIP
├── IP_checkpoints
├── samurai
└── videos
```
